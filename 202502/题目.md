# 螺旋矩阵 - fdu
## 题目描述
Time Limit: 1000 ms

Memory Limit: 256 mb

给定两个正整数 n 和 m ，生成一个 n 行 m 列的顺时针螺旋矩阵（矩阵元素从 1 开始依次递增）。随后有 T 个查询，每个查询包含两个正整数 x 和 y（1 <= x <= n，1 <= y <=m），请你输出该螺旋矩阵中第 x 行第 y 列对应的元素值。

数据范围
- 1 <= n,m <= 1e9
- 1 <= T <= 100
- 1 <= x <= n, 1 <= y <= m

## 输入输出格式
### 输入描述:
- 第一行：两个正整数 n 和 m
- 第二行：一个正整数 T
- 接下来 T 行：每行两个正整数 x 和 y，代表一次查询
### 输出描述:
  对于每个查询，输出一行对应的元素值
##  输入输出样例
  输入样例#:
```
  3 3
  3
  1 1
  2 2
  3 3
```

  输出样例#:
  ```
  1
  9
  5
  ```
