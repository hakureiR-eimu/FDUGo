# BMI 计算与分类

## 题目描述
Time Limit: 1000 ms
Memory Limit: 256 mb
身体质量指数（BMI）是目前国际上常用的衡量人体胖瘦程度以及是否健康的一个标准，其计算公式为：

$\text{BMI} = \frac{\text{体重(kg)}}{\text{身高(m)}^2}$

现给出  个人的身高（以厘米为单位，整数类型）和体重（以千克为单位，浮点数类型），请你计算每个人的BMI值，并根据以下标准判断其体型类型：
- BMI < 18.5：过轻
- 18.5 ≤ BMI < 24：正常
- 24 ≤ BMI < 28：超重
- BMI ≥ 28：肥胖

要求输出每个人的BMI值（保留2位小数）和对应的体型类型。

数据范围
- 1 <= n <= 1000
- 100 <= h <= 250（厘米）
- 20.0 <= w <= 300.0（千克）

输入输出格式
输入描述:
```
第一行输入一个整数 （表示人数）；
接下来 n 行，每行输入一个整数 h（身高，单位：厘米）和一个浮点数 w（体重，单位：千克），两个数用空格分隔。
```
输出描述:
```
对于每个人，输出一行结果，包含两个部分：
1. BMI值（保留2位小数）；
2. 对应的体型类型（过轻/正常/超重/肥胖）。
两个部分用空格分隔。
```
## 输入输出样例
输入样例#:
```
3
170 65.0
160 45.0
180 90.0
```
输出样例#:
```
22.49 正常
17.58 过轻
27.78 超重
```
